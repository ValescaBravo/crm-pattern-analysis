This project explores customer behaviour patterns to identify actionable CRM insights (value concentration, channel preference, lifecycle signals and engagement proxies).

## Business question
What behavioural and lifecycle patterns explain customer value and engagement, and how can we use them to guide CRM actions?

## Approach (summary)
- **EDA:** customer value distribution, RFM signals and engagement indicators
- **Behaviour patterns:** channel preference and activity proxies
- **Lifecycle:** tenure/cohort-style views to understand maturity and retention dynamics
- **Output:** key insights + practical recommendations for CRM planning
